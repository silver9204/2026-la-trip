# 2026 LA 가족여행 모바일 가이드

Google Docs 원본 일정을 모바일에서 빠르게 확인하기 위한 정적 웹앱입니다.

## 개인정보 원칙

- 예약번호, 결제금액, 여권·ESTA 정보는 저장소에 넣지 않습니다.
- 티켓과 예약서는 Google Drive 원본 링크로만 연결합니다.
- 공개 저장소에 PDF, QR 코드, 바코드, 개인 사진을 커밋하지 않습니다.

## GitHub Pages

저장소의 `Settings → Pages`에서 `Deploy from a branch`를 선택하고 `main / (root)`를 게시 소스로 지정합니다.
